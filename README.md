# VowsVibe

## Running the Project

### Prerequisites
- Node.js 18.x or higher
- npm or yarn package manager

### Installation
1. Clone the repository:
```bash
git clone https://github.com/Soumya975p/VowsVibe.git
cd VowsVibe
```

2. Install dependencies:
```bash
npm install
```

### Development Server
Run the development server:
```bash
npm run dev
```

The application will be available at `http://localhost:3000`

### Production Build
Build and run the production version:
```bash
npm run build
npm start
```

### Other Commands
- **Lint**: `npm run lint` - Check code quality
- **Format**: Format code with your preferred formatter
